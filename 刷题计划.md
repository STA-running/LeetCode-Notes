# 刷题计划 · 70 题精刷

> C++ 方向 · 目标岗位为工控/制造业 · 70 题精刷，质量优先

## 核心原则

1. **一题多解**：先想暴力，再想优化，写出来后看题解对比，搞懂最优解的思维
2. **错题本**：做错或想了超过 15 分钟没思路的题，记下来，一周后再做一遍
3. **不追求 hard**：hard 题几乎不考，medium 做到顺滑就够
4. **先理解模式**：同类型的题先集中练（比如连刷 5 道滑动窗口），形成肌肉记忆

## 每日时间安排

```
每天 45-60 分钟：

上午 20 分钟：
  - 复习昨天做错的题，重新默写一遍

晚上 30-40 分钟：
  - 做 1-2 道新题
  - 先自己想 15 分钟，不会再看题解
  - 做完后一定看官方题解或高赞题解，对比思路
```

## 面试前最后一周

1. 把所有标记的错题重刷一遍
2. 练习「说思路」：每道题先口述思路再写代码
3. 熟背四道手写必备题：
   - 反转链表（迭代 + 递归）
   - 二叉树三种遍历（递归 + 迭代）
   - 快速排序（手写 partition）
   - 二分查找（左闭右闭、左闭右开）
