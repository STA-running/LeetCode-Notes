## 学会的三个操作

解决这道题只需要会三个操作：

```cpp
unordered_set<int> n;              // 声明：创建一个哈希集合
n.find(x) != n.end()               // 查找：x 在不在集合里
n.insert(x)                        // 插入：把 x 存进集合
```

### end() 是什么？

```
n.end() = "集合末尾" = "没找到" 的标记

n.find(5) != n.end()   → true  = 找到了（重复了）
n.find(5) == n.end()   → true  = 没找到（第一次见）
```

### 比喻：行李寄存

```
unordered_set<int> n;   ← 空行李架
n.insert(5);             ← 放上箱子 5
n.find(5) != n.end()    ← 5 号箱子在架子上吗？
                         在 → 有人重复寄存了！
                         不在 → 第一次来
```

## 我的通过代码

```cpp
class Solution {
public:
    bool containsDuplicate(vector<int>& nums) {
        unordered_set<int> n;
        int len = nums.size();
        for(int i = 0; i < len; i++){
            if(n.find(nums[i]) != n.end()){
                return true;        // 重复了
            } else {
                n.insert(nums[i]);  // 第一次见，存起来
            }
        }
        return false;               // 全部看完，没重复
    }
};
```

## 扩展：unordered_set 的兄弟姐妹

刷题常用的 STL 容器就四个，记住这个表就够了：

| 容器 | 中文名 | 特点 | 查 | 增 | 删 | 用在哪 |
|------|--------|------|----|-----|----|-------|
| `unordered_set` | 哈希集合 | 不重复、无序、O(1) | `find` | `insert` | `erase` | 判断存在、去重 |
| `unordered_map` | 哈希字典 | 键值对、无序、O(1) | `find` | `insert` / `[]` | `erase` | 统计次数、映射关系 |
| `set` | 有序集合 | 不重复、自动排序 | `find` | `insert` | `erase` | 需要有序去重 |
| `map` | 有序字典 | 键值对、自动排序 | `find` | `insert` / `[]` | `erase` | 需要排序的映射 |

### 最常见的进阶：unordered_map

如果题目问的不是"有没有重复"，而是"每个数字出现了几次"，就要用 `unordered_map`：

```cpp
unordered_map<int, int> count;   // 键=数字，值=次数

// 统计每个数字出现的次数
for (int num : nums) {
    count[num]++;                // 用 [] 直接操作，不存在会自动创建
}

// 或者手动查
for (int num : nums) {
    if (count.find(num) != count.end()) {
        count[num]++;            // 已经出现过，次数+1
    } else {
        count[num] = 1;          // 第一次见
    }
}
```

**核心区别：`unordered_set` 是一个篮子，`unordered_map` 是一个带标签的格子。**

| | unordered_set | unordered_map |
|--|---------------|---------------|
| 存什么 | 单个值（int） | 键值对（key, value） |
| 查找 | `find(key)` | `find(key)` 或 `count[key]` |
| 插入 | `insert(key)` | `insert({key, val})` 或 `count[key] = val` |
| 典型用途 | "出现过吗" | "出现了几次" / "和谁配对" |

### 其他常用操作

```cpp
// 所有容器通用
n.size()          // 元素个数
n.empty()         // 是否为空
n.erase(x)        // 删除元素 x
n.clear()         // 清空全部

// unordered_map 特有
count[5]          // 直接访问键 5 对应的值（不存在则创建默认值）
count.at(5)       // 安全访问，不存在会抛异常
```

## 总结

哈希表相关的 STL 语法不需要背，记住三点就行：

1. **声明**：`unordered_set<类型> 名字;`
2. **查找**：`名字.find(x) != 名字.end()` → "找到了"
3. **插入**：`名字.insert(x)`
