# 链表基础与反转

## 链表长什么样

```
head ──→ [val=1 | next] ──→ [val=2 | next] ──→ [val=3 | next] ──→ nullptr
```

每个节点包含两部分：

- **val**：存储的数据
- **next**：指向下一个节点的指针，最后一个节点指向 `nullptr`

链表和数组最大的区别：**数组是连续内存，链表是分散内存**。所以不能 `list[2]` 直接跳到第三个元素，只能从 `head` 开始，顺着 `next` 一个节点一个节点地走。

## 版本一：通用版本（从零手写，自己管理内存）

适合理解链表原理。所有节点用 `new` 创建，用完 `delete` 销毁。

```cpp
#include <iostream>

// 节点定义
struct Node {
    int val;
    Node* next;
    Node(int x) : val(x), next(nullptr) {}
};

// 链表类
class LinkedList {
private:
    Node* head;

public:
    LinkedList() : head(nullptr) {}

    // 头插：新节点成为新的 head
    void push_front(int x) {
        Node* newNode = new Node(x);
        newNode->next = head;
        head = newNode;
    }

    // 尾插
    void push_back(int x) {
        Node* newNode = new Node(x);
        if (!head) { head = newNode; return; }
        Node* cur = head;
        while (cur->next) cur = cur->next;
        cur->next = newNode;
    }

    // 按值删除
    void remove(int x) {
        if (!head) return;
        if (head->val == x) {
            Node* tmp = head;
            head = head->next;
            delete tmp;
            return;
        }
        Node* cur = head;
        while (cur->next && cur->next->val != x)
            cur = cur->next;
        if (cur->next) {
            Node* tmp = cur->next;
            cur->next = tmp->next;
            delete tmp;
        }
    }

    // 反转链表
    void reverse() {
        Node* prev = nullptr;
        Node* cur = head;
        while (cur) {
            Node* next = cur->next;   // 先保存下一个
            cur->next = prev;         // 指向前一个
            prev = cur;               // prev 前移
            cur = next;               // cur 前移
        }
        head = prev;
    }

    // 遍历打印
    void print() {
        for (Node* cur = head; cur; cur = cur->next)
            std::cout << cur->val << " -> ";
        std::cout << "nullptr\n";
    }

    // 析构：释放所有节点内存
    ~LinkedList() {
        Node* cur = head;
        while (cur) {
            Node* next = cur->next;
            delete cur;
            cur = next;
        }
    }
};

// 使用示例
int main() {
    LinkedList list;
    list.push_back(1);
    list.push_back(2);
    list.push_back(3);
    list.print();        // 1 -> 2 -> 3 -> nullptr

    list.reverse();
    list.print();        // 3 -> 2 -> 1 -> nullptr

    list.remove(2);
    list.print();        // 3 -> 1 -> nullptr
}
```

## 版本二：力扣版本（只用 ListNode*，不用管释放）

力扣已经帮你定义好了 `ListNode`，传进来的链表你不需要自己构建，也不需要释放内存。**你只需要操作指针。**

### 节点定义

```cpp
struct ListNode {
    int val;
    ListNode *next;
    ListNode() : val(0), next(nullptr) {}
    ListNode(int x) : val(x), next(nullptr) {}
    ListNode(int x, ListNode *next) : val(x), next(next) {}
};
```

三种构造函数：

| 写法 | 结果 |
|------|------|
| `new ListNode()` | val=0, next=nullptr |
| `new ListNode(5)` | val=5, next=nullptr |
| `new ListNode(5, node)` | val=5, next=node |

### 手动构建链表（刷题时测试用）

```cpp
// 用 vector 快速构建链表
ListNode* buildList(const vector<int>& nums) {
    ListNode dummy(0);           // 哑节点，避免空头判断
    ListNode* tail = &dummy;
    for (int x : nums) {
        tail->next = new ListNode(x);
        tail = tail->next;
    }
    return dummy.next;
}
```

> **哑节点（dummy node）** 技巧：创建一个临时头节点，让它的 `next` 指向真正的头。这样就不需要单独处理 `head == nullptr` 或者插入到头部的情况，最后返回 `dummy.next` 即可。

### 反转链表——迭代版（力扣 206）

```cpp
class Solution {
public:
    ListNode* reverseList(ListNode* head) {
        ListNode* prev = nullptr;
        ListNode* cur = head;
        while (cur) {
            ListNode* next = cur->next;  // 1. 保存下一个
            cur->next = prev;             // 2. 反转指向
            prev = cur;                   // 3. prev 前进
            cur = next;                   // 4. cur 前进
        }
        return prev;                      // 新的头
    }
};
```

### 反转链表——递归版（面试加分）

```cpp
class Solution {
public:
    ListNode* reverseList(ListNode* head) {
        if (!head || !head->next) return head;
        ListNode* newHead = reverseList(head->next);
        head->next->next = head;   // 把 head 接到反转后的末尾
        head->next = nullptr;
        return newHead;
    }
};
```

递归版理解难点：`head->next->next = head` 意思是**让 head 的下一个节点反过来指向 head**。画个图：

```
原始：  head → [2 → 3 → nullptr]
                     ↑
               递归返回后变成了：
                head → [2 ← 3]
                       ↓
                      nullptr? 不对——是 3 指向 2

递归思路：先反转 head 后面的部分，
         然后把 head 接到反转结果的末尾
```

### 力扣实战技巧

| 操作 | 写法 | 说明 |
|------|------|------|
| 判空 | `if (!head) return nullptr;` | head 为 null 时直接返回 |
| 判单个 | `if (!head->next) return head;` | 只有一个节点，反转就是它自己 |
| 遍历 | `for (auto p = head; p; p = p->next)` | 标准遍历写法 |
| 取下一个 | `auto next = cur->next;` | 永远先保存再改指针 |
| 哑节点 | `ListNode dummy(0); dummy.next = head;` | 避免空头判断 |

## 版本三：STL 版本（直接用标准库）

C++ 标准库提供 `forward_list`（单向链表）和 `list`（双向链表）。但**面试和刷题几乎不用**，因为：

1. 链表面试考的就是手玩指针，用 STL 封装了等于没考
2. `forward_list` 没有 `push_back`，只有 `push_front`，用起来不顺手
3. 刷题 90% 的场景用 `vector` 就够了

```cpp
#include <forward_list>
#include <iostream>

int main() {
    std::forward_list<int> list;

    // 头插（没有 push_back）
    list.push_front(1);   // 1
    list.push_front(2);   // 2 -> 1
    list.push_front(3);   // 3 -> 2 -> 1

    // 遍历
    for (int x : list)
        std::cout << x << " ";   // 3 2 1

    // 反转
    list.reverse();              // 1 -> 2 -> 3

    // 删除指定值
    list.remove(2);              // 1 -> 3

    // 排序（链表归并排序）
    list.sort();                 // 1 -> 3
}
```

如果需要双向操作，用 `std::list`：

```cpp
std::list<int> list;
list.push_back(1);     // 尾插
list.push_front(0);    // 头插
list.pop_back();       // 删尾
list.pop_front();      // 删头
```

## 刷题建议

### 先从反转链表开始

反转链表是**链表题的基础操作**，后面的环形链表、回文链表、反转链表 II 全都是它的变形。先把迭代版的三个步骤练到肌肉记忆：

```
1. next = cur->next   存下一个
2. cur->next = prev   反转指向
3. prev = cur; cur = next  一起前移
```

### 链表和 vector 的选择

| 场景 | 用什么 |
|------|--------|
| 刷力扣链表题（反转、环、相交） | `ListNode*` |
| 日常算法题（排序、查找） | `vector` |
| 需要频繁头插尾插 | `deque` |
| 需要中间插入删除 | `list`（但刷题极少用） |
