# 矩阵操作与旋转

## 旋转矩阵：转置 + 反转

顺时针旋转 90° = **先转置，再每行反转**。

### 转置（沿主对角线翻转）

```
(i, j) → (j, i)
```

```cpp
for (int i = 0; i < n; i++)
    for (int j = i; j < n; j++)
        swap(matrix[i][j], matrix[j][i]);
```

### 每行反转（水平翻转）

```
(j, i) → (j, n-1-i)
```

```cpp
for (int i = 0; i < n; i++)
    reverse(matrix[i].begin(), matrix[i].end());
```

### 为什么等于旋转？

想象一张透明纸：
1. **沿对角线对折** → 行变列，列变行（转置）
2. **水平翻页** → 左右互换（反转）

两步合起来就是顺时针旋转 90°。

## 拓展：其他旋转

**逆时针旋转 90°** = 先转置，再垂直翻转（上下翻）：

```cpp
for (int i = 0; i < n; i++)
    for (int j = i; j < n; j++)
        swap(matrix[i][j], matrix[j][i]);

for (int i = 0; i < n / 2; i++)
    swap(matrix[i], matrix[n - 1 - i]);
```

**旋转 180°** = 先上下翻转，再左右翻转。

## 注意事项

- 方阵才能用这个公式（n×n）
- 转置时内层循环 `j = i` 而不是 `j = 0`，否则会重复 swap 两遍等于没做
- 原地操作，不需要额外矩阵，空间 O(1)
